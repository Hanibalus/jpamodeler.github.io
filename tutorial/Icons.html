Installing JPA Modeler Plugins :

To get the plugins ready for installation, start NetBeans as described in "Installing and the Starting the NetBeans IDE." Start NetBeans and select the "Tools > Plugins" menu item
Browse NBM
Be sure to remove any existing installation of the JPA Modeler plugin. Failure to do so will most likely cause installation / execution problems. Switch to the "Downloaded" tab and select the "Add Plugins..." button to browse to the downloaded nbm file.
Once the file has been selected, NetBeans will display module information as follows:
click on Install
Now select the "Install" button at the bottom :
When the NetBeans IDE Installer screen is displayed, click Next.
  • Accept the license terms and click Install.
  • If additional Validation screens appear, including a confirmation to install an unsigned plugin. Just select "Continue" until you get to the final screen.
The plug-in has now been installed and can be seen in the "Installed" tab:
Click Finish to restart NetBeans. 

Verifying Your Installation:

Once you have installed the JPA Modeler Plugins into NetBeans, its installation can further be confirmed via the "New File > Persistence > JPA Diagram file"; if it shows up, the installation seems to have gone ok:
The JPA Modeler is now ready for you to work with, to create a new project and develop code.
